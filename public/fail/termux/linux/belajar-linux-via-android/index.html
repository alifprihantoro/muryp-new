    Belajar Linux via Android

    Home/ Halaman Blog / Belajar Linux via Android
        Welcome dude! bagi kalian yang baru mampir dan salam juga bagi kalian para pembaca setia blog ini hehehe. Kali ini Saya ingin memberikan informasi kepada kalian. Bagi kalian yang mengikuti step step di blog ini, kalian pasti sudah mengerti apa itu termux. Bagi kalian yang belum mengetahui jelas mengenai termux, kalian bisa kunjungi artikel disini
       OK saya anggap kalian sudah paham dengan termux. Kalian bisa menggunakan beberapa kode dibawah ini untuk membantu kalian belajar termux. Karena kita akan menggunakan termux sebagai terminalnya dan beberapa aplikasi lainnya.
    Pertama kalian install AnLinux, keyboard Hacker, dan VNC Viewer di Play Store atau klick pada link tersebut yang sudah mengarah ke play store.
    1. Setelah itu kalian buka AnLinux dan ikuti gambar berikut:
      1. klick pada menu dasboard
      2. Pilih Lingkungan yang akan dipilih, saya sarankan ubuntu terlebih dahulu karena tutorial mengenai ubuntu lebih banyak.
      3. klick copy lalu launch
    2. Kita akan dibawa ke aplikasi Termux
      • sebelumnya ketik 
    apt update && apt upgrade 
      •  setelah itu pastekan kode yang kita copy tadi 
      •  Setelah itu kita kembali ke AnLinux dan klick dasboard. 
      •  Lalu klick dan pilih Ubuntu lalu klick GUI pilih aja yang XTE. 
      •  Setelah itu copy dan Launch 
      •  Kita akan kembali ke Termux lagi. Sebelum pastekan, pastikan kalian sudah masuk ke ubuntu. JIka belum  ketik kode berikut :

    ./start-ubuntu.sh 

      •  Lalu klick enter. Sebenarnya sampai disini kita sudah menginstall ubuntu. Namun kalo cuman terminal saja kan gimana. Jadi kode tersebut untuk menampilkan ubuntu versi desktop di HP kita. 
      • pastekan kode tadi. NB: maaf pada gambar saya salah menuliskan.  Tunggu sampai selesai. Pilih bahasa dan masukkan password 2 kali. NB : saat memasukkan password memang tak terlihat. Pilih Y/N pilih N saja
      • Keluar dari Termux namun jangan di keluarkan(kunci aplikasi) 
      •  Masuk ke VNC Viewer lalu klick icon tambah. 
      •  setelah itu masukkan IP dan Nama

    Localhost:1  

    nama terserah

      •  Lalu klick OK lalu klick Connect 
      •  Jika muncul seperti ini abaikan saja
      •  Ketikkan password yang kalian buat tadi 
      •  Klick ok. Tada... sudah selesai menginstall Ubuntu di Android
    Untuk Keyboard Hacker sendiri digunakan untuk tombol/mirip keyboard fisik sehingga memudahkan untuk melakukan shortcut.

    Ok untuki selanjutnya, mari kita belajara bersama mengenai LInux ini. Jika ada kesusahan atau pertanyaan, silahkan komentar dibawah. Terima kasih untuk kunjungannya. Jika anda tertarik dengan artikel di blog ini kalian bisa follow media sosial saya.
