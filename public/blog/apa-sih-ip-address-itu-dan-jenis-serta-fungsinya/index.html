    IP addres sering kita jumpai namun tidak diketahui. Lalu apa sih IP Address Itu? Dan Jenis Serta Fungsinya apa? Oke mari kita simak berikut ini.

    Apa Sih IP Address Itu? Dan Jenis Serta Fungsinya

    Apa itu IP?

    Secara sederhana IP address sangat mirip dengan nomor telepon, dimana satu sama lain memiliki nomor yang unik seperti layaknya nomor hp kita. Dan nomor IP tersebut digunakan sebagai identitas di internet. Tahukah, jika HP pintar yang anda pakai juga memiliki IP address juga. Dan biasanya IP digunakan untuk menghubungkan perangkat satu dengan yang lainnya. 

    Contohnya handphone kita saat berinternet membutuhkan sebuah link dimana link tersebut sebenarnya sekumpulan angka IP yang dimiliki oleh server. 


    Lalu Siapa yang mencatat/mengelola IP?

    Internet Assigned Numbers Authority atau disingkat IANA lah yang bertanggung jawab untuk masalah root DNS, pemberian alamat IP, dan referensi protokol Internet lainnya. IANA mendelegasikan otoritas untuk semua domain (non-root) ke registri lainnya. Sehingga tidak ada IP publick yang sama. 


    Tahukah anda Smart phone yang kalian pakai sekarang ini pada jaman dulu sudah setara komputer segede gaban.


    Lalu fungsinya apa?

    Seperti sudah disinggung diatas IP address bertujuan untuk identitas masing-masing komputer. Meskipun IP jarang kita temui, sebenarnya IP sering kita pakai lho. Namun kita sering sekali menggunakan link untuk mepermudahnya. Contohnya jika kalian salin 172.217.194.101: dan pastekan ke url maka kalian akan diarahkan ke google.com atau mesin pencari google. 


    IPv4 vs IPv6

    Saat ini sudah miliaran website dan perangkat terhubung ke internet. Karena itulah dibuatlah IPv6 untuk memenuhi kuota tersebut. Dimana IPv4 yang terdiri dari angka 32 bit sudah hampir habis sehingga dibuatlah IPv6 yang memiliki panjang 128 bit. 

    Dalam IP yang memiliki panjang 32bit, memiliki 4 buah baris angka yang dipisahkan oleh titik. Yang mempresentasikan desimal 8 digit biner. Satu baris tersebut juga disebut oktet. Dan rentang IPv4 adalah dari 0.0.0.0 sampai 255.255.255.255. Dan untuk IPv6 bisa mencapai 340.282.366.920.938.463.463.374.607.431.768.211.456 alamat. 


    Perbedaan IP publik dan privat


    Perbedaan yang pasti ialah mirip seperti namanya. Dimana IP publik digunakan untuk internet dan diatur oleh IANA sedangkan IP Privat hanya digunakan di lokal saja. Namun jika anda ingin IP Privat bisa diakses diinternet kalian membutuhkan jaringan internet untuk menghubungkannya. Meskipun untuk melakukannya biasanya kita bisa beli server/hosting di tempat jasa penyedia hosting. Pernahkah kalian mendengar istilah Hosting, VPS, VPN, FTP jika belum silahkan klik disini. 


    IP Dinamis vs IP Statis

    Perbedaannya ialah hanya terletak pada perubahan IP yang dilakukan oleh IP Dinamis. Dimana seperti kita ketahui IPv4 sudah tidak bisa menampung lebih banyak IP, sehingga para penyedia internet memberikan IP yang sering berubah setiap seminggu, sebulan, bahkan setahun sekali. Merestart router juga mengganti alat IP publik kita. 

    Untuk IP statis sendiri kebalikannya. Dimana IP tersebut dibutuhkan sebagai server web, Hosting, VPN dan FTP. Karena dibutuhkan IP yang tetap untuk layanannya.



    Shared IP VS Dedicated IP

    Perbedaannya ialah jika shared IP dipakai rame-rame yang biasanya ada di Shared Hosting maka dedicated IP dipakai oleh satu domain saja. Meskipun begutu beberapa penyedia layanan shared hosting memperbolehkan untuk memakai dedicated IP
